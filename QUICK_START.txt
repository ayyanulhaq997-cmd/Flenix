============================================
FENIX FLUTTER APP - QUICK START (Windows)
============================================

THREE SIMPLE STEPS:

STEP 1: Install Flutter SDK (One-time only)
========================================
1. Download Flutter: https://flutter.dev/docs/get-started/install/windows
2. Extract to: C:\flutter
3. Add C:\flutter\bin to Windows PATH
4. Restart PowerShell

Verify: flutter --version

STEP 2: Copy Flutter Project to Windows PC
==========================================
Copy these files from Replit to your Windows:
  From: /home/runner/workspace/fenix-flutter/
  To:   C:\Flutter-Projects\fenix-flutter\

Files to copy:
  - lib/ (entire folder)
  - pubspec.yaml
  - SETUP.md

STEP 3: Run Setup Script
=======================
1. Save this file to: C:\Flutter-Projects\fenix-flutter\setup.bat
2. Double-click setup.bat
3. Wait for app to open in Chrome
4. Login with: admin@fenix.local / Admin@123456

That's it! App should be running.
